# Roadmap

## MVP (v0.1.0)
- CLI pipeline for local HTML/MD input
- Stable anchors + clean Markdown output per page
- Generate `llms.txt` and `llms-full.txt`
- Content-addressed manifest
- Basic eval harness with report

## Post-MVP
- Better boilerplate detection and removal
- Optional sitemap crawler
- JSON-LD schema validation
- Richer eval scoring with golden datasets
